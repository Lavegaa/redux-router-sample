# redux-router-sample
리덕스와 라우트를 사용하기 위한 템플릿을 만들었다.
# 빠른시작

## 클론

```
git clone https://github.com/Lavegaa/redux-router-sample.git
```

## dependencies 설치

```
npm install
```

or

```
yarn
```

## 시작

```
npm start
```

or

```
yarn start
```

# 라이브러리
- react-redux
- redux
- redux-actions
- redux-devtools-extension
- react-router-dom
- immer
- styled-components
- node-sass
- classnames
